## Step-by-Step Setup

1. Migrate Database.
   ```bash
   php artisan migrate
   ```
   jika error gunakan ini
    ```bash
   php artisan migrate:fresh
   ```
2. Masukan dummy data dengan menggunakan ini.
   ```bash
   nphp artisan db:seed
   ```
3. kemudian jalankan laravel.
   ```bash
   php artisan serve
   ```
